# Typeset Markdown Test Folder

Copy this folder into `Documents/.typeset` to test the app with a broad Markdown fixture set.

Suggested copy target:

```text
C:\Users\Ethan Rodrigues\Documents\.typeset\markdown-kitchen-sink
```

Files included:

- `Kitchen-Sink.md`: broad CommonMark and GFM syntax coverage.
- `Project.md` plus `Project/Child-Note.md`: Typeset parent note and sub-note structure.
- `Research/Tables-And-Tasks.md`: tables, task lists, alignment, escaped pipes.
- `Research/Links-And-Media.md`: inline links, reference links, autolinks, images.
- `Research/Code-Blocks.md`: fenced code, indented code, Mermaid as code, math as code.
- `Research/Unsafe-HTML.md`: HTML and script-like content to confirm sanitization.
- `Research/Edge-Cases.md`: whitespace, escapes, nested lists, long lines.

Do not copy `LAYOUT.md` files from here; Typeset should generate them.
